# VibeEdit AI

[![Build Flutter APK](https://github.com/praaatap/VibeEdit-Ai/actions/workflows/flutter_build.yml/badge.svg)](https://github.com/praaatap/VibeEdit-Ai/actions/workflows/flutter_build.yml)

Professional AI-Powered Video Editing via your Smartphone.

## Project Structure

- **vibeedit**: Flutter Frontend Application
- **vibeedit-backend**: Python FastAPI Backend Service

## Features

- **AI Magic Tools**: Auto-cut, Smart Remove, AI Enhance
- **Professional Editing**: Multi-track timeline, effects, transitions
- **Cloud Export**: Optimized for Instagram, TikTok, YouTube
